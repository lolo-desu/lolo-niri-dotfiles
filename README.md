# lolo-niri-dotfiles

Personal dotfiles snapshot for niri, waybar, and fcitx5 (Matugen macOS-dark theme).

## Contents
- `~/.config/niri/`
- `~/.config/waybar/`
- `~/.local/share/fcitx5/themes/macOS-dark-matugen/`
- `~/.config/matugen/templates/fcitx5-macos-dark/`

## Notes
- These files are copies from the live system.
- Apply changes manually or with symlinks as you prefer.
